# DID BTC1 JS

Monorepo for did:btc1 js/ts implementation and supporting packages.

## Installation

* Clone monorepo

```sh
git clone https://github.com/jintekc/did-btc1-js.git
cd did-btc1-js
```

* Install dependencies

```sh
pnpm install
```

* Build

```sh
pnpm build
```
